# MVP Roadmap

## v0.1 Goals

- ingest station and ADA status data
- create first-pass catchments
- join tract-level disability, age, and poverty indicators
- compute a basic accessibility gap score
- export map-friendly outputs
- ship one borough walkthrough notebook

## v0.1 Non-Goals

- perfect routing realism on day one
- fully polished public dashboard UX
- long historical outage coverage if it slows the first release too much

## Stretch Goals

- network-based isochrone catchments
- historical outage logging and reliability scoring
- council district and community district rollups
- public-facing static reporting outputs

## Release Philosophy

The first release should prove the analytical frame clearly: reliable accessibility is different from nominal accessibility, and that difference can be measured with public data.
